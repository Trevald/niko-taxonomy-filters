Adds taxonomy filters to post listing pages in admin. Creates dropdown only for taxonomies connected to current post type.


== Installation ==


1. Upload the plugin folder files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Does this work woth custom post types? =

Yes.

= Does this work with custom taxonomies? =

Yes.
